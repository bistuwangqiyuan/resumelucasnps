## 📞 使用说明

### 开发环境

```bash
# 克隆项目
git clone <repo-url>
cd resumelucasnps

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

### 内容更新

**更新奖项**:
1. 编辑 `app/achievements/page.jsx`
2. 在 `achievements` 数组中添加新奖项
3. 将证书图片放入 `image/` 文件夹

**更新作品**:
1. 编辑 `app/works/page.jsx`
2. 在 `works` 数组中添加新项目

**更新诗歌**:
1. 编辑 `poem.md` 文件
2. 按格式添加新诗歌

详见 `DEPLOYMENT.md` 文档
