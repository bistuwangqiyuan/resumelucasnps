# 王一然个人简历网站 - 产品需求文档 (PRD)

## 项目概述

**项目名称**: 王一然个人简历网站  
**项目类型**: 个人展示网站  
**目标用户**: 学校师生、评委、访客  
**技术栈**: Next.js 14, React, Tailwind CSS  
**部署平台**: Netlify  
**开发日期**: 2025年10月  

## 项目目标

1. **展示个人成就**: 全面展示王一然同学的学业成绩、奖项荣誉和特长才能
2. **作品集展示**: 展示编程作品和诗歌创作
3. **自荐宣传**: 用于学校评选和个人品牌建设
4. **技术能力证明**: 通过网站本身展示AI编程能力

## 核心功能

### 1. 首页 (/)
**功能描述**: 作为网站的门面，提供概览和导航

**核心元素**:
- 动态粒子背景效果
- 个人头像/标识
- 个人标语和标签
- 快速统计数据（奖项数、诗歌数、项目数等）
- 精选成就卡片
- 导航按钮

**技术实现**:
- Canvas实现粒子背景
- Intersection Observer实现滚动动画
- 响应式网格布局

### 2. 关于我页面 (/about)
**功能描述**: 详细介绍个人信息和背景

**核心元素**:
- 基本信息卡片（姓名、学校、班级等）
- 家庭背景介绍
- 完整自荐信
- 五大自荐理由
- 学业成绩展示

**数据来源**: resume.md, intruduction.md

### 3. 奖项成就页面 (/achievements)
**功能描述**: 展示所有获奖记录

**核心元素**:
- 统计数据（国家级/省部级/校级分类）
- 时间线展示
- 证书图片画廊
- 分类筛选
- 获奖故事

**数据来源**: resume.md, image/ 文件夹

**技术实现**:
- Next/Image优化图片加载
- 时间线组件
- 图片懒加载

### 4. 我的作品页面 (/works)
**功能描述**: 展示编程项目和技能

**核心元素**:
- 项目列表（植物大战僵尸、个人网站等）
- 项目详情（描述、技术栈、特性、链接）
- 技能水平图表
- 编程理念介绍

**外部链接**: 
- 植物大战僵尸: https://lako.top

**技术实现**:
- 动态技能条
- 项目卡片组件
- 外部链接处理

### 5. 诗歌集页面 (/poems)
**功能描述**: 展示63篇诗歌作品

**核心元素**:
- 诗歌统计数据
- 诗歌列表/网格展示
- 创作理念介绍
- 诗歌分类

**数据来源**: poem.md

**技术实现**:
- 服务器端数据读取
- Markdown解析（如需要）
- 分页或虚拟滚动（优化性能）

## 设计规范

### 配色方案
**主色调**: 科技感渐变
- 青色 (Cyan): #06b6d4
- 蓝色 (Blue): #3b82f6
- 紫色 (Purple): #8b5cf6
- 粉色 (Pink): #ec4899

**辅助色**:
- 背景深色: #030712 → #111827
- 文字白色: #ffffff
- 文字灰色: #9ca3af

**语义色**:
- 成功/绿色: #10b981
- 警告/橙色: #f97316
- 错误/红色: #ef4444

### 排版规范
- **标题**: 
  - H1: 3xl-6xl (48-72px)
  - H2: 2xl-4xl (32-48px)
  - H3: xl-2xl (24-32px)
- **正文**: base-lg (16-18px)
- **小字**: sm-xs (12-14px)

**字体**:
- 系统默认字体栈，优先显示中文字体
- 代码字体: monospace

### 动效规范
- **过渡时间**: 300ms (标准), 150ms (快速), 500ms (慢速)
- **缓动函数**: ease-out (默认), ease-in-out (平滑)
- **动画类型**:
  - 淡入淡出 (Fade)
  - 缩放 (Scale)
  - 滑动 (Slide)
  - 粒子效果 (Particle)

### 响应式断点
- **xs**: < 640px (手机竖屏)
- **sm**: 640px (手机横屏)
- **md**: 768px (平板)
- **lg**: 1024px (小屏笔记本)
- **xl**: 1280px (桌面)
- **2xl**: 1536px (大屏)

## 组件架构

### 布局组件
1. **Header**: 导航栏，固定顶部，支持移动端折叠菜单
2. **Footer**: 页脚，包含版权信息和链接
3. **Layout**: 根布局，包含Header和Footer

### 功能组件
1. **ParticleBackground**: Canvas粒子背景
2. **AnimatedSection**: 滚动触发动画容器
3. **ImageWithFallback**: 带加载状态和错误处理的图片组件

### 页面组件
- **page.jsx**: 各页面主组件
- **loading.jsx**: 加载状态
- **not-found.jsx**: 404页面

## 技术规范

### 代码规范
- ESLint配置
- 使用函数组件和Hooks
- 组件文件使用 .jsx 扩展名
- 使用 export default 导出主组件

### 性能优化
1. **图片优化**: 
   - 使用 Next/Image
   - 配置 unoptimized: true (Netlify静态部署)
   - 懒加载

2. **代码分割**: 
   - 动态导入重组件
   - App Router 自动代码分割

3. **SEO优化**:
   - 完善的metadata
   - robots.txt
   - sitemap.xml
   - 语义化HTML

### 浏览器兼容
- 现代浏览器 (Chrome, Firefox, Safari, Edge)
- 移动浏览器 (iOS Safari, Chrome Mobile)
- 不支持 IE11
